## Set
- Set is an `Interface`
- Set don't store duplicate values
- HashSet, TreeSet & LinkedHashSet are Classes that implement set.
- We can easily use `for-each` loop in set
- we can pass set in the constructor of an arraylist and it will copy all the data of the set.
- > ```ArrayList\<Integer> arr = new ArrayList<>(set_name);```
---
### HashSet
- Data stored in random order
- Operation in Constant time.
- It is like arraylist but it doesn't store duplicate values
